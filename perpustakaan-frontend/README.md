# Perpustakaan Frontend

React application untuk Sistem Manajemen Perpustakaan.

## Setup
```bash
npm install
npm run dev
```

## Features

- Login & Register
- CRUD Categories
- CRUD Books
- CRUD Borrowings
- Automatic stock management
- Responsive design

## Tech Stack

- React 18
- React Router DOM
- Axios
- Tailwind CSS
